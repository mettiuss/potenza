# Potenza

Discord bot for the Official Fortnite Italia server.

## Run Locally
Clone the project
```bash
  git clone https://github.com/mettiuss/potenza.git
```

Go to the project directory
```bash
  cd potenza
```

Start the bot
```bash
  npm run test
```
